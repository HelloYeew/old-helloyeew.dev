Rūrusetto (ルールセット)

Rūrusetto is a project that I made it for containing all osu! rulesets source and all information about osu! rulesets (a.k.a osu! rulesets wiki). The idea is from me and my friend (who is making an osu! rulesets) want to build a wiki that contain all osu! rulesets so you can download it from one place and find a new rulesets to install and play easily. Currently, I always use all of my free time to write a new entire codebase for this website since the old codebase is hard to add a new function and the progress is now more than a half is complete!

The screenshot that you are seeing is a draft version of a new website homepage that I am rewriting it.
